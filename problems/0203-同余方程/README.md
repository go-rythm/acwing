## [203\. 同余方程](https://www.acwing.com/problem/content/205/)

### 题目

求关于 $x$ 的同余方程 $ax ≡ 1 \\pmod b$ 的最小正整数解。

### 输入格式

输入只有一行，包含两个正整数 $a,b$，用一个空格隔开。

### 输出格式

输出只有一行，包含一个正整数 $x$，表示最小正整数解。

输入数据保证一定有解。

### 数据范围

$2 \\le a,b \\le 2 \\times 10^9$

### 输入样例：

```
3 10
```

### 输出样例：

```
7
```
