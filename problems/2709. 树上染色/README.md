## [2709. 树上染色](https://www.acwing.com/problem/content/2711/)

### 题目

有一棵点数为 *N* 的树，树边有边权。

给你一个在 *0 ~ N* 之内的正整数 *K*，你要在这棵树中选择 *K* 个点，将其染成黑色，并将其他的 *N-K* 个点染成白色。

将所有点染色后，你会获得黑点两两之间的距离加上白点两两之间距离的和的收益。

问收益最大值是多少。

### 输入格式

第一行两个整数 *N,K*。

接下来 *N-1* 行每行三个正整数 *fr,to,dis*，表示该树中存在一条长度为 *dis* 的边 *(fr,to)*。

输入保证所有点之间是联通的。

### 输出格式

输出一个正整数，表示收益的最大值。

### 数据范围

*1 ≤ N ≤ 2000*,

*0 ≤ K ≤ N*

### 输入样例：

```
5 2
1 2 3
1 5 1
2 3 1
2 4 2
```

### 输出样例：

```
17
```

### 样例解释

将点 *1,2* 染黑就能获得最大收益。
