## [1430. 贝特西之旅](https://www.acwing.com/problem/content/1432/)

### 题目

一个正方形城镇被划分为了 *N × N* 个方格区域。

农场位于左上角方格内，市场位于左下角方格内。

贝特西想要来一场城镇之旅，它计划从农场出发，经过每个方格区域一次，最终到达市场完成旅行。

下图是 *N = 3* 时，贝特西的一种旅行方案：

```
----------------
|    |    |    |
| F**********  |
|    |    | *  |
------------*---
|    |    | *  |
|  *****  | *  |
|  * | *  | *  |
---*---*----*---
|  * | *  | *  |
|  M | ******  |
|    |    |    |
----------------
```

给定整数 *N*，请你计算共有多少种不同的旅行路线。

### 输入格式

一个整数 *N*。

### 输出格式

一个整数，表示旅行路线总数。

### 数据范围

*1 ≤ N ≤ 7*

### 输入样例：

```
3
```

### 输出样例：

```
2
```
