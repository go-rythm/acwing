## [1311. 组合](https://www.acwing.com/problem/content/1313/)

### 题目

LMZ 有 *n* 个不同的朋友，他每天晚上要选 *m* 个朋友一起聚会，而且要求每天晚上的选择都不一样。

那么 LMZ 能够持续多少个这样的夜晚呢？

当然，LMZ 的一年有 *10007* 天，所以他想知道答案 *bmod 10007* 的值。

### 输入格式

第一行一个整数 *t*，表示有 *t* 组数据；

接下来 *t* 行，每行两个整数 *n,m*。

### 输出格式

共 *t* 行，每行一个数，为 *C_n^m bmod 10007* 的答案。

### 数据范围

*1 ≤ t ≤ 200*,

*1 ≤ m ≤ n ≤ 2 × 10^8*

### 输入样例：

```
4
5 1
5 2
7 3
4 2
```

### 输出样例：

```
5
10
35
6
```
