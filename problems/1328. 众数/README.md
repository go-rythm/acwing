## [1328. 众数](https://www.acwing.com/problem/content/1330/)

### 题目

输入 *n* 个数，问哪个数出现的次数最多。

如果有多个出现次数最多的数，输出最大的那个数。

### 输入格式

第一行有一个正整数 *n*。

第二行有 *n* 个正整数，表示每一个数，相邻两个数之间用一个空格隔开。

### 输出格式

只输出一个数即答案。

### 数据范围

保证 *n* 和输入的 *n* 个正整数都不会超过 *2048*。

### 输入样例：

```
7
2 3 6 2 3 6 7
```

### 输出样例：

```
6
```
