## [3457. 两小时学完C语言](https://www.acwing.com/problem/content/3460/)

### 题目

知乎上有个宝宝问：“两个小时内如何学完 *C* 语言？”当然，问的是“学完”并不是“学会”。

假设一本 *C* 语言教科书有 *N* 个字，这个宝宝每分钟能看 *K* 个字，看了 *M* 分钟。

还剩多少字没有看？

### 输入格式

输入在一行中给出 *3* 个正整数，分别是 *N*，教科书的总字数；*K*，是宝宝每分钟能看的字数；*M*，是宝宝看书的分钟数。

题目保证宝宝看完的字数不超过 *N*。

### 输出格式

在一行中输出宝宝还没有看的字数。

### 数据范围

*1 ≤ N ≤ 4 × 10^5*,

*1 ≤ K ≤ 3000*,

*1 ≤ M ≤ 120*

### 输入样例：

```
100000 1000 72
```

### 输出样例：

```
28000
```
