## [1931. 奶牛冰壶](https://www.acwing.com/problem/content/1933/)

### 题目

奶牛冰壶是哞林匹克运动会中一项流行的寒冷天气运动。

像常规的冰壶运动一样，这项运动包括两支队伍，每支队伍都在一片冰上滑过 *N* 块重石头。

比赛结束时，冰上共有 *2N* 块石头，每块都位于不同的二维平面的点上。

但是，奶牛冰壶的计分规则有些奇怪。

如果一块石头位于由三个对手的石头作为顶点构成的三角形的内部（边界也算内部），则该石头被称作“已捕获”。

一支队伍的得分等于其捕获对方石头的数量。

给定 *2N* 个石头的具体位置，请确定两支队伍分别的得分。

### 输入格式

第一行包含整数 *N*。

接下来 *N* 行，每行包含两个整数 *x,y*，表示第一支队伍的一块石头的横纵位置坐标。

再接下来 *N* 行，每行包含两个整数 *x,y*，表示第二支队伍的一块石头的横纵位置坐标。

### 输出格式

输出第一支队伍和第二支队伍的得分。

### 数据范围

*3 ≤ N ≤ 50000*,

*-40000 ≤ x,y ≤ 40000*

### 输入样例：

```
4
0 0
0 2
2 0
2 2
1 1
1 10
-10 3
10 3
```

### 输出样例：

```
1 2
```

### 样例解释

第一支队伍捕获 *(1,1)* 处的石头。

第二支队伍捕获 *(0,2)* 和 *(2,2)* 处的石头。
