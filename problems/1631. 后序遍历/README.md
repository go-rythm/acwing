## [1631. 后序遍历](https://www.acwing.com/problem/content/1633/)

### 题目

假设二叉树上各结点的权值互不相同且都为正整数。

给定二叉树的前序遍历和中序遍历，请你输出二叉树的后序遍历的第一个数字。

### 输入格式

第一行包含整数 *N*，表示二叉树结点总数。

第二行给出二叉树的前序遍历序列。

第三行给出二叉树的中序遍历序列。

### 输出格式

输出二叉树的后序遍历的第一个数字。

### 数据范围

*1 ≤ N ≤ 50000*

### 输入样例：

```
7
1 2 3 4 5 6 7
2 3 1 5 4 7 6
```

### 输出样例：

```
3
```
