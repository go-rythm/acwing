## [3453. 整数查询](https://www.acwing.com/problem/content/3456/)

### 题目

给定若干个可能很大的正整数，请你计算它们相加的和。

### 输入格式

最多不超过 *100* 个正整数，每个数的长度不超过 *100*。

每个整数占一行。

最后输入一行一个 *0*，表示输入结束。

注意整数可能有前导 *0*。

### 输出格式

输出所有数的和。

### 输入样例：

```
123456789012345678901234567890
123456789012345678901234567890
123456789012345678901234567890
0
```

### 输出样例：

```
370370367037037036703703703670
```
