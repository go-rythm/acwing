## [3590. 数字之和](https://www.acwing.com/problem/content/3593/)

### 题目

给定整数 *n*，请你计算 *n* 的各位数字之和以及 *n^2* 的各位数字之和。

### 输入格式

输入包含多组测试数据。

每组数据占一行，包含一个整数 *n*。

### 输出格式

每组数据输出一行，两个整数，表示 *n* 的各位数字之和以及 *n^2* 的各位数字之和，整数之间单个空格隔开。

### 数据范围

输入最多包含 *100* 组数据，

*0 < n < 40000*。

### 输入样例：

```
4
12
97
39999
```

### 输出样例：

```
4 7
3 9
16 22
39 36
```
