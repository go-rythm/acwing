## [3692. 最长连续公共子序列](https://www.acwing.com/problem/content/3695/)

### 题目

输入两个字符串 *s_1,s_2*。

输出最长连续公共子串长度和最长连续公共子串。

### 输入格式

一行，两个字符串 *s_1,s_2*，用空格隔开。

### 输出格式

第一行输出最长连续公共子串的长度

第二行输出最长连续公共子串。如果不唯一，则输出最后一个。

### 数据范围

*1 ≤ |s_1|,|s_2| ≤ 100*

### 输入样例：

```
abcdefg qwercdefiok
```

### 输出样例：

```
4
cdef
```
