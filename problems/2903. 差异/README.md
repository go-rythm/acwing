## [2903. 差异](https://www.acwing.com/problem/content/2906/)

### 题目

一个长度为 *n* 的字符串 *S*，令 *T_i* 表示它从第 *i* 个字符开始的后缀。求

**sum_{1le i < jle n}len(T_i)+len(T_j)-2timeslcp(T_i,T_j)**

其中，*len(a)* 表示字符串 *a* 的长度，*lcp(a,b)* 表示字符串 *a* 和字符串 *b* 的最长公共前缀。

### 输入格式

一行，一个字符串 *S*。

### 输出格式

一行，一个整数，表示所求值。

### 数据范围

*2 ≤ n ≤ 5 × 10^5*,

*S* 由小写英文字母组成。

### 输入样例：

```
cacao
```

### 输出样例：

```
54
```
