## [1070. 括号配对](https://www.acwing.com/problem/content/1072/)

### 题目

Hecy 又接了个新任务：BE 处理。

BE 中有一类被称为 GBE。

以下是 GBE 的定义：

- 空表达式是 GBE
- 如果表达式 A 是 GBE，则 [A] 与 (A) 都是 GBE
- 如果 A 与 B 都是 GBE，那么 AB 是 GBE

下面给出一个 BE，求至少添加多少字符能使这个 BE 成为 GBE。

注意：BE 是一个仅由 `(`、 `)`、 `[`、 `]` 四种字符中的若干种构成的字符串。

### 输入格式

输入仅一行，为字符串 BE。

### 输出格式

输出仅一个整数，表示增加的最少字符数。

### 数据范围

对于所有输入字符串，其长度小于100。

### 输入样例：

```
[])
```

### 输出样例：

```
1
```
