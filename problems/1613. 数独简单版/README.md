## [1613. 数独简单版](https://www.acwing.com/problem/content/1615/)

### 题目

数独是一种传统益智游戏，你需要把一个 *9 × 9* 的数独补充完整，使得图中每行、每列、每个 *3 × 3* 的九宫格内数字 *1 ~ 9* 均恰好出现一次。

请编写一个程序填写数独。

### 输入格式

输入共 *9* 行，每行包含一个长度为 *9* 的字符串，用来表示数独矩阵。

其中的每个字符都是 *1 ~ 9* 或 *.*（表示尚未填充）。

### 输出格式

输出补全后的数独矩阵。

数据保证有唯一解。

### 输入样例：

```
.2738..1.
.1...6735
.......29
3.5692.8.
.........
.6.1745.3
64.......
9518...7.
.8..6534.
```

### 输出样例：

```
527389416
819426735
436751829
375692184
194538267
268174593
643217958
951843672
782965341
```
