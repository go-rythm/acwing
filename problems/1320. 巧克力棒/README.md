## [1320. 巧克力棒](https://www.acwing.com/problem/content/1322/)

### 题目

TBL 和 X 用巧克力棒玩游戏。

每次一人可以从盒子里取出若干条巧克力棒，或是将一根取出的巧克力棒吃掉正整数长度。

TBL 先手两人轮流，无法操作的人输。

他们以最佳策略一共进行了 *10* 轮（每次一盒）。

你能预测胜负吗？

### 输入格式

输入数据共 *20* 行。

第 *2i−1* 行一个正整数 *N_i* ，表示第 *i* 轮巧克力棒的数目。

第 *2i* 行 *N_i* 个正整数 *L_{i,j}* ，表示第 *i* 轮巧克力棒的长度。

### 输出格式

输出数据共 *10* 行。每行输出 YES 或 NO，表示 TBL 是否会赢。

如果胜则输出 NO，否则输出 YES。

### 数据范围

*1 ≤ N ≤ 14*,

*1 ≤ L ≤ 10^9*

### 输入样例：

```
3

```

