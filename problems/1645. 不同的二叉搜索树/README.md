## [1645. 不同的二叉搜索树](https://www.acwing.com/problem/content/1647/)

### 题目

给定一个整数 *n*，求以 *1 … n* 为节点组成的二叉搜索树有多少种？

结果对 *10^9 + 7* 取模后输出。

### 输入格式

共一行，包含一个整数 *n*。

### 输出格式

输出一个整数，表示对 *10^9 + 7* 取模后的结果。

### 数据范围

*1 ≤ n ≤ 1000*

### 输入样例：

```
3
```

### 输出样例：

```
5
```

### 样例解释

当 *n = 3* 时, 一共有 *5* 种不同结构的二叉搜索树:

```
1         3     3      2      1
           /     /      /       
     3     2     1      1   3      2
    /     /                        
2     1         2                 3
```
