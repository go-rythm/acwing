## [2965. 有意义的字符串](https://www.acwing.com/problem/content/2968/)

### 题目

*B* 君有两个好朋友，他们叫宁宁和冉冉。

有一天，冉冉遇到了一个有趣的题目：输入 *b,d,n*，求

**bigg lfloor left ( frac{b+sqrt{d}}{2} right ) ^n bigg rfloor bmod 7528443412579576937**

### 输入格式

一行三个整数 *b,d,n*。

### 输出格式

一行一个数表示模 *7528443412579576937* 之后的结果。

### 数据范围

*0 < b^2 ≤ d ≤ 10^{18}*,

*n ≤ 10^{18}*，

*b bmod 2=1*,

*d bmod 4=1*

### 输入样例：

```
1 5 9
```

### 输出样例：

```
76
```
