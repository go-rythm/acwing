## 803. 区间合并

### 题目

给定 n 个区间 [l<sub>i</sub>, r<sub>i</sub>]，要求合并所有有交集的区间。

注意如果在端点处相交，也算有交集。

输出合并完成后的区间个数。

例如：[1,3]和[2,6]可以合并为一个区间[1,6]。

### 输入格式
第一行包含整数n。

接下来n行，每行包含两个整数l和r。

### 输出格式
共一行，包含一个整数，表示合并区间完成后的区间个数。

### 数据范围
1≤n≤100000,
−10^9≤l<sub>i</sub>≤r<sub>i</sub>≤10^9

### 输入样例：

```
5
1 2
2 4
5 6
7 8
7 9
```

### 输出样例：

```
3
```

### 题解

按左端点排序（若相同则按右端点排序）后，

两个区间只有两种关系：

1. 扩充现有区间
2. 保存当前区间到结果中，建立一个新区间
