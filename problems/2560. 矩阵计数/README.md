## [2560. 矩阵计数](https://www.acwing.com/problem/content/2562/)

### 题目

一个 *N × M* 的方格矩阵，每一个方格中包含一个字符 `O` 或者字符 `X`。

要求矩阵中不存在连续一行 *3* 个 `X` 或者连续一列 *3* 个 `X`。

问这样的矩阵一共有多少种？

### 输入格式

输入一行包含两个整数 *N* 和 *M*。

### 输出格式

输出一个整数代表答案。

### 数据范围

*1 ≤ N,M ≤ 5*

### 输入样例：

```
2 3
```

### 输出样例：

```
49
```
