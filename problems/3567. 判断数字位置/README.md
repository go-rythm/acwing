## [3567. 判断数字位置](https://www.acwing.com/problem/content/3570/)

### 题目

给定一个由字母和数字构成的字符串，请你判断字符串中数字的位置。

位置坐标从 *1* 开始。

### 输入格式

第一行包含整数 *T*，表示共有 *T* 组测试数据。

每组数据占一行，包含一个字符串。

### 输出格式

每组数据输出一行数字，用空格隔开，按顺序表示字符串中出现的数字的位置。

### 数据范围

*1 ≤ T ≤ 10*,

给定字符串长度不超过 *50*。

### 输入样例：

```
1
a3b4c5
```

### 输出样例：

```
2 4 6
```
