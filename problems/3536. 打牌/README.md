## [3536. 打牌](https://www.acwing.com/problem/content/3539/)

### 题目

牌只有 *1* 到 *9*，手里拿着已经排好序的牌 *a*，对方出牌 *b*，用程序判断手中牌是否能够压过对方出牌。

规则：出牌牌型有 *5* 种：

- “一张”，如 *4*，则 *5…9* 可压过。
- “两张”，如 *44*，则 *55,66,77,…,99* 可压过。
- “三张”，如 *444*，规则如“两张”。
- “四张”，如 *4444*，规则如“两张”。
- “五张”，牌型只有 *12345,23456,34567,45678,56789* 五种，后面的比前面的均大。

### 输入格式

第一行包含字符串 *a*，表示手中的牌。

第二行包含字符串 *b*，表示对方出的牌。

### 输出格式

若能压过输出 `YES`，否则 `NO`。

### 数据范围

*a* 的长度不超过 *100*,

*b* 保证合法。

### 输入样例：

```
12233445566677
33
```

### 输出样例：

```
YES
```
