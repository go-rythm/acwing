## [3569. 三角形相加](https://www.acwing.com/problem/content/3572/)

### 题目

写一个 CTriangle 类，要求可以接受 *CTriangle(y,x)* 形式的构造，表示含义如下：*A(0,y) B(0,0) C(x,0)*。

要求能够处理若干个三角形的相加（点 *B* 保持不变，两直角边相加）。

### 输入格式

输入有若干行，每行两个数 *y,x*，读到 *0* 表示结束。

### 输出格式

输出一行表示三点坐标，格式参见样例。

### 数据范围

输入不超过 *100* 对数对。

*1 ≤ x,y ≤ 100*

### 输入样例：

```
10 20
1 31
0
```

### 输出样例：

```
A(0,11),B(0,0),C(51,0)
```
