## [3392. 递推数列](https://www.acwing.com/problem/content/3395/)

### 题目

给定 *a_0,a_1*，以及 *a_n=p × a_{n-1} + q × a_{n-2}* 中的 *p,q*。

这里 *n ≥ 2*。

求第 *k* 个数 *a_k* 对 *10000* 的模。

### 输入格式

输入包括 *5* 个整数：*a_0、a_1、p、q、k*。

### 输出格式

第 *k* 个数 *a_k* 对 *10000* 的模。

### 数据范围

*1 ≤ a_0,a_1,p,q,k ≤ 10000*

### 输入样例：

```
20 1 1 14 5
```

### 输出样例：

```
8359
```
