## [889\. 满足条件的01序列](https://www.acwing.com/problem/content/891/)

### 题目

给定 $n$ 个 $0$ 和 $n$ 个 $1$，它们将按照某种顺序排成长度为 $2n$ 的序列，求它们能排列成的所有序列中，能够满足任意前缀序列中 $0$ 的个数都不少于 $1$ 的个数的序列有多少个。

输出的答案对 $10^9+7$ 取模。

### 输入格式

共一行，包含整数 $n$。

### 输出格式

共一行，包含一个整数，表示答案。

### 数据范围

$1 \\le n \\le 10^5$

### 输入样例：

```
3
```

### 输出样例：

```
5
```
