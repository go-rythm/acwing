## [3649. 成绩的等级](https://www.acwing.com/problem/content/3652/)

### 题目

给出一百分制成绩，要求输出成绩等级 `A`、 `B`、 `C`、 `D`、 `E`。

*90* 分以上为 *A*，*80 ~ 89* 分为 *B*，*70 ~ 79* 分为 *C*，*60 ~ 69* 分为 *D*，*60* 分以下为 *E*。

### 输入格式

一个整数 *score*。

### 输出格式

一个字符，表示成绩等级。

### 数据范围

*0 ≤ score ≤ 100*

### 输入样例：

```
90
```

### 输出样例：

```
A
```
