## [793\. 高精度乘法](https://www.acwing.com/problem/content/795/)

### 题目

给定两个非负整数（不含前导 $0$） $A$ 和 $B$，请你计算 $A \\times B$ 的值。

### 输入格式

共两行，第一行包含整数 $A$，第二行包含整数 $B$。

### 输出格式

共一行，包含 $A \\times B$ 的值。

### 数据范围

$1 \\le A 的长度 \\le 100000$,

$0 \\le B \\le 10000$

### 输入样例：

```
2
3
```

### 输出样例：

```
6
```
