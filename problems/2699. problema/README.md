## [2699. problema](https://www.acwing.com/problem/content/2701/)

### 题目

一次考试共有 *n* 个人参加，第 *i* 个人说：“有 *a_i* 个人分数比我高，*b_i* 个人分数比我低。”

问最少有几个人没有说真话(可能有相同的分数)。

### 输入格式

第一行一个整数 *n*。

接下来 *n* 行每行两个整数，第 *i+1* 行的两个整数分别代表 *a_i、b_i*。

### 输出格式

一个整数，表示最少有几个人没有说真话。

### 数据范围

*1 ≤ n ≤ 10^5*,

*0 ≤ a_i,b_i ≤ n*

### 输入样例：

```
3
2 0
0 2
2 2
```

### 输出样例：

```
1
```
