## [3398. 升次方](https://www.acwing.com/problem/content/3401/)

### 题目

已知三个 **正整数** *a,n,m* 之间满足 *n = a^m*。

现在，给定 *n* 和 *m*，请你计算 *a* 的值。

### 输入格式

共一行，包含两个正整数 *n* 和 *m*。

### 输出格式

输出一个正整数，表示 *a* 的值。

### 数据范围

*1 ≤ n ≤ 10^{70000}*,

*2 ≤ m ≤ 5*,

数据保证有解。

### 输入样例：

```
8 3
```

### 输出样例：

```
2
```
