## [1586. 连续因子](https://www.acwing.com/problem/content/1588/)

### 题目

在一个正整数 *N* 的所有因子中，可能存在几个连续的数字。

例如，*630* 可以分解为 *3×5×6×7*，其中 *5、6、7* 是三个连续的数字。

现在给定任意正整数 *N*，请你找到最大连续因子数，并列出连续因子的最小序列。

### 输入格式

一个正整数 *N*。

### 输出格式

第一行输出最大连续因子数。

第二行以 `factor[1]*factor[2]*...*factor[k]` 的格式输出连续因子的最小序列。

因子按升序排序，不包括 *1*。

### 数据范围

*1 < N < 2^{31}*

### 输入样例：

```
630
```

### 输出样例：

```
3
5*6*7
```
