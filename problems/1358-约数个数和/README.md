## [1358\. 约数个数和](https://www.acwing.com/problem/content/1360/)

### 题目

设 $d(x)$ 为 $x$ 的约数个数，给定 $N,M$，求

$$\\sum\_{i=1}^N\\sum\_{j=1}^Md(ij)$$

### 输入格式

输入多组测试数据。

第一行，一个整数 $T$，表示测试数据的组数。

接下来的 $T$ 行，每行两个整数 $N、M$。

### 输出格式

$T$ 行，每行一个整数，表示你所求的答案。

### 数据范围

$1 \\le N,M,T \\le 50000$

### 输入样例：

```
2
7 4
5 6
```

### 输出样例：

```
110
121
```
