## [3192. 出现次数最多的数](https://www.acwing.com/problem/content/3195/)

### 题目

给定 *n* 个正整数，找出它们中出现次数最多的数。

如果这样的数有多个，请输出其中最小的一个。

### 输入格式

输入的第一行只有一个正整数 *n*，表示数字的个数。

输入的第二行有 *n* 个整数 *s_1, s_2, …, s_n*。

相邻的数用空格分隔。

### 输出格式

输出这 *n* 个次数中出现次数最多的数。

如果这样的数有多个，输出其中最小的一个。

### 数据范围

*1 ≤ n ≤ 1000*,

*1 ≤ s_i ≤ 10000*

### 输入样例：

```
6
10 1 10 20 30 20
```

### 输出样例：

```
10
```
