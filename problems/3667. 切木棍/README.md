## [3667. 切木棍](https://www.acwing.com/problem/content/3670/)

### 题目

你有一根长度为 *n* 的木棍(*n* 是整数)，现在需要你用这个木棍做一个矩形（但是不能是正方形），矩形的每条边的长度为整数。

你可以将这个木棍切成四份，作为矩形的四条边，请问你组成多少种不同的矩形？

### 输入格式

输入包含多组测试数据。

每组数据占一行，包含一个整数 *n*。

### 输出格式

每组数据输出一行结果，表示可以组成的矩形数。

### 数据范围

*1 ≤ n ≤ 10^5*

### 输入样例：

```
6
20
```

### 输出样例：

```
1
4
```
