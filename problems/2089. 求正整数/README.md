## [2089. 求正整数](https://www.acwing.com/problem/content/2091/)

### 题目

对于任意输入的正整数 *n*，请编程求出具有 *n* 个不同因子的最小正整数 *m*。

例如：*n=4*，则 *m=6*，因为 *6* 有 *4* 个不同整数因子 *1，2，3，6*；而且是最小的有 *4* 个因子的整数。

### 输入格式

包含一个整数 *n*。

### 输出格式

输出一个整数 *m*。

### 数据范围

*1 ≤ n ≤ 50000*

### 输入样例：

```
4
```

### 输出样例：

```
6
```
