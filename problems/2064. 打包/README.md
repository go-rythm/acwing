## [2064. 打包](https://www.acwing.com/problem/content/2066/)

### 题目

皮普有 *N* 个字符串，每个字符串中只包含 *A* 到 *Z* 的大写字母。

皮普希望将它们打包成若干组，每组恰好包含 *K* 个字符串。

每个字符串都正好属于一个组。

一个组的分数定义为该组内所有字符串的最长公共前缀的长度，例如：

- 组 `{RAINBOW, RANK, RANDOM, RANK}` 的分数为 *2*（最长公共前缀为 `RA`）。
- 组 `{FIRE, FIREBALL, FIREFIGHTER}` 的分数为 *4*（最长公共前缀为 `FIRE`）。
- 组 `{ALLOCATION, PLATE, WORKOUT, BUNDLING}` 的分数为 *0*（最长公共前缀为空）。

请帮助皮普合理分组，使得各组的分数之和最大化。

### 输入格式

第一行包含整数 *T*，表示共有 *T* 组测试数据。

对于每组数据，第一行包含两个整数 *N* 和 *K*。

接下来 *N* 行，每行包含一个皮普的字符串。

### 输出格式

每组数据输出一个结果，每个结果占一行。

结果表示为 `Case #x: y`，其中 *x* 为组别编号（从 *1* 开始），*y* 为各组的分数的最大可能值。

### 数据范围

*1 ≤ T ≤ 100*,

*2 ≤ N ≤ 10^5*,

*2 ≤ K ≤ N*,

保证 *N* 能被 *K* 整除。

字符串长度不为空，每组测试数据的所有字符串包含的字母总数量不超过 *2 × 10^6*。

### 输入样例1：

```
2
2 2
KICK
START
8 2
G
G
GO
GO
GOO
GOO
GOOO
GOOO
```

### 输出样例1：

```
Case #1: 0
Case #2: 10
```

### 输入样例2：

```
1
6 3
RAINBOW
FIREBALL
RANK
RANDOM
FIREWALL
FIREFIGHTER
```

### 输出样例2：

```
Case #1: 6
```
