## [197. 阶乘分解](https://www.acwing.com/problem/content/199/)

### 题目

给定整数 *N*，试把阶乘 *N!* 分解质因数，按照算术基本定理的形式输出分解结果中的 *p_i* 和 *c_i* 即可。

### 输入格式

一个整数 *N*。

### 输出格式

*N!* 分解质因数后的结果，共若干行，每行一对 *p_i, c_i*，表示含有 *p_i^{c_i}* 项。按照 *p_i* 从小到大的顺序输出。

### 数据范围

*1 ≤ N ≤ 10^6*

### 输入样例：

```
5
```

### 输出样例：

```
2 3
3 1
5 1
```

### 样例解释

*5! = 120 = 2^3 * 3 * 5*
