## [1778. 牛为什么过马路](https://www.acwing.com/problem/content/1780/)

### 题目

牛为什么过马路？

我们可能永远不知道全部的原因，但是可以肯定的是，农夫约翰的奶牛确实经常过马路。

实际上，它们过马路的频率很高，以至于当它们横穿马路时，经常会撞到彼此，这是农夫约翰想要纠正它们的地方。

约翰饲养了 *N* 个品种的奶牛，他的每一块田专门用于放牧一个特定品种的奶牛。

例如，用于饲养品种 *12* 奶牛的田地只能用于饲养品种 *12* 的奶牛，而不用于饲养其他品种的奶牛。

一条很长的马路贯穿了他的农场。

路的一侧有一排 *N* 块田地（每种品种一个），另一侧也有一排 *N* 块田地（每种品种一个）。

因此，当一头奶牛希望从一块属于其品种的田地走到另一块属于其品种的田地时，就需要过马路。

如果约翰的计划更周密一些，按照同样的品种排列顺序安置两侧的田地，这样就可以使得同一品种的两块田地隔着马路彼此正对。

那么，奶牛在过马路时，就不用担心与不同品种的奶牛发生碰撞。

不幸的是，两侧田地的安置顺序可能不太一样。

因此，约翰发现可能会存在两个品种的田地之间交叉的情况。

如果两个品种 *A* 的田地之间的任何路径都一定与两个品种 *B* 的田地之间的任何路径相交，则称这一对不同品种 *(A,B)* 处于交叉状态。

约翰希望尽可能的减少交叉品种对的数量。

出于管理组织方面的考虑，他认为可以将牛在道路一侧循环移动，这也就相当于这一侧的田地发生了循环移位。

对于 *0 ≤ k < N*，每头奶牛都向后移动 *k* 位，最后 *k* 位的奶牛移动到最前面 *k* 位。

例如，如果道路一侧的田地开始按品种排序为 *3、7、1、2、5、4、6*，并经历 *k=2* 的循环移位，则新的顺序为 *4、6、3、7、1、2、5*。

现在，请你计算，当一侧田地进行适当的循环移位以后，交叉品种对的数量最少是多少。

### 输入格式

第一行包含整数 *N*。

接下来 *N* 行，按品种 ID 描述了道路一侧的田地的排列顺序。

再接下来 *N* 行，按品种 ID 描述了道路另一侧的田地的排列顺序。

品种 ID 从 *1* 到 *N*。

### 输出格式

输出经过适当循环移位后，交叉品种对的最少数量。

### 数据范围

*1 ≤ N ≤ 10^5*

### 输入样例：

```
5
5
4
1
3
2
1
3
2
5
4
```

### 输出样例：

```
0
```
