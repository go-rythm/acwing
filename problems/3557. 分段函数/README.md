## [3557. 分段函数](https://www.acwing.com/problem/content/3560/)

### 题目

编写程序，计算下列分段函数 *y=f(x)* 的值。

- 当 *0 ≤ x < 2* 时，*y= -x+2.5*；
- 当 *2 ≤ x < 4* 时，*y=2-1.5(x-3)(x-3)*；
- 当 *4 ≤ x < 6*，*y=x/2-1.5*。

### 输入格式

第一行包含整数 *T*，表示共有 *T* 组测试数据。

每组数据占一行，包含一个整数 *x*。

### 输出格式

每组数据输出一行，一个结果，格式为 `y=k`，其中 *k* 为计算得到的值，保留一位小数。

### 数据范围

*0 ≤ x ≤ 5*

### 输入样例：

```
2
1
3
```

### 输出样例：

```
y=1.5
y=2.0
```
