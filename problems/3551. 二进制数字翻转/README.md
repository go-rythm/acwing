## [3551. 二进制数字翻转](https://www.acwing.com/problem/content/3554/)

### 题目

给定一个整数 *n*，将其二进制位反转（共 *32* 位），然后输出对应的十进制数。

### 输入格式

第一行包含整数 *T*，表示共有 *T* 组测试数据。

每组数据占一行，包含一个整数 *n*。

### 输出格式

每组数据输出一行，一个结果。

### 数据范围

*1 ≤ T ≤ 100*,

*0 < n < 2^{32}*,

**注意 *n* 的取值范围**。

### 输入样例：

```
2
2
3
```

### 输出样例：

```
1073741824
3221225472
```
