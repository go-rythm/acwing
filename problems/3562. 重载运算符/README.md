## [3562. 重载运算符](https://www.acwing.com/problem/content/3565/)

### 题目

建立一个角类，在这个类中重载减号运算符（角度相减），并实现求出角度的正弦值的函数。

### 输入格式

第一行包含整数 *T*，表示共有 *T* 组测试数据。

每组数据占一行包含两个整数表示角度。

### 输出格式

每组数据输出一行，一个结果，表示两角相减的正弦值，保留小数点后两位。

### 数据范围

*1 ≤ T ≤ 100*,

角度取值范围 *[-360,360]*。

### 输入样例：

```
1
60 30
```

### 输出样例：

```
0.50
```
