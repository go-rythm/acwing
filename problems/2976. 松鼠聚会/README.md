## [2976. 松鼠聚会](https://www.acwing.com/problem/content/2979/)

### 题目

有 *N* 个小松鼠，它们的家用一个点 *(x,y)* 表示，两个点的距离定义为：点 *(x,y)* 和它周围的 *8* 个点即上下左右四个点和对角的四个点，距离为 *1*。

现在 *N* 个松鼠要走到一个松鼠家去聚会，求需要走过的最短距离。

### 输入格式

第一行给出数字 *N*，表示有多少只小松鼠。

下面 *N* 行，每行给出 *x,y* 表示其家的坐标。

### 输出格式

表示为了聚会走的路程和最小为多少。

### 数据范围

*0 ≤ N ≤ 10^5*,

*-10^9 ≤ x,y ≤ 10^9*

### 输入样例：

```
6
-4 -1
-1 -2
2 -4
0 2
0 3
5 -2
```

### 输出样例：

```
20
```
