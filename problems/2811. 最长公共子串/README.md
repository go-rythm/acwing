## [2811. 最长公共子串](https://www.acwing.com/problem/content/2813/)

### 题目

给定 *n* 个字符串，试求出这些字符串的最长公共子串。

### 输入格式

第一行一个整数 *n*。

下面第 *2* 到 *n+1* 行，每行一个字符串。

### 输出格式

仅一行，包含一个正整数，表示 *n* 个字符串的最长公共子串长度。

### 数据范围

*2 ≤ n ≤ 11*,

每个字符串的长度不超过 *10000*，

字符串均由小写字母构成。

### 输入样例：

```
2
ababc
cbaab
```

### 输出样例：

```
2
```
