## [838\. 堆排序](https://www.acwing.com/problem/content/840/)

### 题目

输入一个长度为 $n$ 的整数数列，从小到大输出前 $m$ 小的数。

### 输入格式

第一行包含整数 $n$ 和 $m$。

第二行包含 $n$ 个整数，表示整数数列。

### 输出格式

共一行，包含 $m$ 个整数，表示整数数列中前 $m$ 小的数。

### 数据范围

$1 \\le m \\le n \\le 10^5$，

$1 \\le 数列中元素 \\le 10^9$

### 输入样例：

```
5 3
4 5 1 3 2
```

### 输出样例：

```
1 2 3
```
