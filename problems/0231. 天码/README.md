## [231. 天码](https://www.acwing.com/problem/content/233/)

### 题目

给定你一个由 *N* 个不同整数构成的整数序列，从这个整数序列中选出 *4* 个数，使得这 *4* 个数的唯一公约数为 *1*。

求满足条件的四元组的个数。

### 输入格式

输入中包含多组测试用例。

每个测试用例占据两行，第一行包含整数 *N*。

第二行包含 *N* 个用空格隔开的整数（均不超过 *10000*），表示完整的整数序列。

### 输出格式

每个测试用例输出一个结果，每个结果占一行。

### 数据范围

*1 ≤ N ≤ 10000*

### 输入样例：

```
4
2 3 4 5
4
2 4 6 8
7
2 3 4 5 7 6 8
```

### 输出样例：

```
1
0
34
```
