## [3684. a与b得到c](https://www.acwing.com/problem/content/3687/)

### 题目

给出 *a,b,c*（*3* 个整数），判断 *a,b* 能否通过 `+-*/` 得到 *c*，*ab* 可以交换位置，可以输出 `YES`，不可以则输出 `NO`。

### 输入格式

输入包含多组测试数据。

每组数据占一行，包含三个整数 *a,b,c*。

### 输出格式

每组数据输出一行结果。

### 数据范围

*1 ≤ a,b,c ≤ 10^9*

### 输入样例：

```
3 8 2
4 8 2
```

### 输出样例：

```
NO
YES
```
