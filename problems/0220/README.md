## [220\. 最大公约数](https://www.acwing.com/problem/content/222/)

### 题目

给定整数 $N$，求 $1 \\le x,y \\le N$ 且 $GCD(x,y)$ 为素数的数对 $(x,y)$ 有多少对。

$GCD(x,y)$ 即求 $x，y$ 的最大公约数。

### 输入格式

输入一个整数 $N$。

### 输出格式

输出一个整数，表示满足条件的数对数量。

### 数据范围

$1 \\le N \\le 10^7$

### 输入样例：

```
4
```

### 输出样例：

```
4
```
