## [3685. 求众数](https://www.acwing.com/problem/content/3688/)

### 题目

给定一个长度为 *n* 的整数序列，请你求出该序列的众数。

众数就是一个序列中出现次数最多的数字。

如果不唯一，则输出小的那个值。

### 输入格式

第一行输入一个整数 *n*，表示有 *n* 个数。

第二行输入 *n* 个整数。

### 输出格式

输出序列中的众数，如果不唯一，则输出小的那个值。

### 数据范围

*1 ≤ n ≤ 10^5*,

序列中整数的取值范围 *[-10^9,10^9]*。

### 输入样例：

```
8
10 3 8 8 3 2 2 2
```

### 输出样例：

```
2
```
