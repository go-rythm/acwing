## [2617. 最大土地面积](https://www.acwing.com/problem/content/2619/)

### 题目

在某块平面土地上有 *N* 个点，你可以选择其中的任意四个点，将这片土地围起来，当然，你希望这四个点围成的多边形面积最大。

### 输入格式

第 *1* 行一个正整数 *N*。

接下来 *N* 行，每行 *2* 个数 *x,y*，表示该点的横坐标和纵坐标。

### 输出格式

最大的多边形面积，答案精确到小数点后 *3* 位。

### 数据范围

*4 ≤ N ≤ 2000*,

*|x|,|y| ≤ 10^5*

### 输入样例：

```
5
0 0
1 0
1 1
0 1
0.5 0.5
```

### 输出样例：

```
1.000
```
