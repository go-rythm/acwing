## [886\. 求组合数 II](https://www.acwing.com/problem/content/888/)

### 题目

给定 $n$ 组询问，每组询问给定两个整数 $a，b$，请你输出 $C\_a^b \\bmod (10^9 + 7)$ 的值。

### 输入格式

第一行包含整数 $n$。

接下来 $n$ 行，每行包含一组 $a$ 和 $b$。

### 输出格式

共 $n$ 行，每行输出一个询问的解。

### 数据范围

$1 \\le n \\le 10000$,

$1 \\le b \\le a \\le 10^5$

### 输入样例：

```
3
3 1
5 3
2 2
```

### 输出样例：

```
3
10
1
```
