## [2525. 区间内逆序对数量](https://www.acwing.com/problem/content/2527/)

### 题目

给定一个长度为 *n* 的序列 *a_1,a_2,…,a_n*。

现在要对该序列进行 *m* 次询问操作。

每次询问给定两个整数 *l,r (l < r)*，请你求出 *a_l,…,a_r* 之间的逆序对数量。

### 输入格式

第一行包含两个整数 *n,m*。

第二行包含 *n* 个整数，表示 *a_1,a_2,…,a_n*。

接下来 *m* 行，每行包含两个整数 *l,r*，表示一次询问。

### 输出格式

共 *m* 行，每行输出一次询问的答案。

### 数据范围

*1 ≤ n,m ≤ 10^5*,

*0 ≤ a_i ≤ 10^9*,

*1 ≤ l < r ≤ n*

### 输入样例：

```
5 2
3 5 4 1 2
1 3
2 5
```

### 输出样例：

```
1
5
```
