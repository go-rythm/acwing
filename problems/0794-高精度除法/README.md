## [794\. 高精度除法](https://www.acwing.com/problem/content/796/)

### 题目

给定两个非负整数（不含前导 $0$） $A，B$，请你计算 $A / B$ 的商和余数。

### 输入格式

共两行，第一行包含整数 $A$，第二行包含整数 $B$。

### 输出格式

共两行，第一行输出所求的商，第二行输出所求余数。

### 数据范围

$1 \\le A的长度 \\le 100000$,

$1 \\le B \\le 10000$,

$B$ 一定不为 $0$

### 输入样例：

```
7
2
```

### 输出样例：

```
3
1
```
