## [1942. 假期计划](https://www.acwing.com/problem/content/1944/)

### 题目

博维尼亚航空公司运营的航班连接了奶牛赖以生存的 *N* 个农场，编号 *1 ~ N*。

其中 *1 ~ K* 号农场被指定为枢纽。

目前，共有 *M* 个单程航班连接这些农场，其中航班 *i* 从农场 *u_i* 飞往农场 *v_i*，花费为 *d_i* 元。

该航空公司最近收到了 *Q* 个单程旅行的请求，其中第 *i* 个请求是从农场 *a_i* 前往农场 *b_i*。

为了从 *a_i* 到达 *b_i*，旅行可以包含任何直飞航班（甚至可以多次经过同一农场），但是必须至少经过一个枢纽（起点和终点也算）。

这个要求可能会使得一部分旅行请求无法被满足。

但是，对于所有可满足的旅行请求，请你确定满足这些旅行请求所需花费的最低总费用。

### 输入格式

第一行包含四个整数 *N,M,K,Q*。

接下来 *M* 行，每行包含整数 *u_i,v_i,d_i*。

接下来 *Q* 行，每行包含两个整数 *a_i,b_i*。

### 输出格式

第一行输出可满足的旅行请求的数量。

第二行输出这些旅行请求所需花费的最低总费用。

### 数据范围

*1 ≤ N ≤ 200*,

*1 ≤ K ≤ 100,K ≤ N*,

*1 ≤ M,Q ≤ 10000*,

*1 ≤ d_i ≤ 10^6*

### 输入样例：

```
3 3 1 3
3 1 10
1 3 10
1 2 7
3 2
2 3
1 2
```

### 输出样例：

```
2
24
```

### 样例解释

*3 to 2* 只有一条可行路线，花费 *10+7*，*2 to 3* 没有可行路线，*1 to 2* 只有一条可行路线，花费 *7*。
