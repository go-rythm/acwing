## [3682. 图的连通分量](https://www.acwing.com/problem/content/3685/)

### 题目

给定 *m* 个 **不同的** 整数 *a_1,a_2,…,a_m*，每个数都处于 *[0,2^n-1]* 的范围内。

一个图按照如下约束建立：

如果整数 *X* 和 *Y* 满足 *X & Y=0*（*&* 是按位与运算），则 *X* 和 *Y* 由一条无向边连接。

请计算，图中连通分量的数量。

### 输入格式

输入包含多组测试数据。

每组数据第一行包含两个整数 *n,m*。

第二行包含 *m* 个整数 *a_1,a_2,…,a_m*。

### 输出格式

每组数据输出一行结果，表示连通分量的个数。

### 数据范围

*0 ≤ n ≤ 22*,

*1 ≤ m ≤ 2^n*,

*0 ≤ a_i < 2^n*

### 输入样例：

```
2 3
1 2 3
5 5
5 19 10 20 12
5 6
5 19 10 20 12 0
```

### 输出样例：

```
2
2
1
```
