## [3121. 可爱的质数](https://www.acwing.com/problem/content/3124/)

### 题目

给定一个质数 *p*，以及一个整数 *b*，一个整数 *n*，现在要求你计算一个最小的非负整数 *l*，满足 *b^l equiv n pmod p*。

### 输入格式

仅一行，有 *3* 个整数，依次代表 *p, b, n*。

### 输出格式

仅一行，如果有 *l* 满足该要求，输出最小的非负整数 *l*，否则输出 `no solution`。

### 数据范围

*2 ≤ b,n < p < 2^{31}*

### 输入样例：

```
5 2 3
```

### 输出样例：

```
3
```
