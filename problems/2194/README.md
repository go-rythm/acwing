## [2194\. 负载平衡问题](https://www.acwing.com/problem/content/2196/)

### 题目

$G$ 公司有 $n$ 个沿铁路运输线环形排列的仓库，每个仓库存储的货物数量不等。

如何用最少搬运量可以使 $n$ 个仓库的库存数量相同。

搬运货物时，只能在相邻的仓库之间搬运。

数据保证一定有解。

### 输入格式

第 $1$ 行中有 $1$ 个正整数 $n$，表示有 $n$ 个仓库。

第 $2$ 行中有 $n$ 个正整数，表示 $n$ 个仓库的库存量。

### 输出格式

输出最少搬运量。

### 数据范围

$1 \\le n \\le 100$,

每个仓库的库存量不超过 $100$。

### 输入样例：

```
5
17 9 14 16 4
```

### 输出样例：

```
11
```
