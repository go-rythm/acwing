## [3552. 发财数](https://www.acwing.com/problem/content/3555/)

### 题目

一个大于等于 *2* 的整数，如果可以分解为 *8* 个或 *8* 个以上的素数相乘，则称其为发财数，请你输出第 *n* 个发财数。

### 输入格式

第一行包含整数 *t*，表示共有 *t* 组测试数据。

每组数据一行，一个整数 *n*。

### 输出格式

每组数据输出一行，一个结果。

### 数据范围

*1 ≤ t ≤ 1000*,

*1 ≤ n ≤ 10000*

### 输入样例：

```
2
1
2
```

### 输出样例：

```
256
384
```
