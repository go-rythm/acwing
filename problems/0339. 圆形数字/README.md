## [339. 圆形数字](https://www.acwing.com/problem/content/341/)

### 题目

定义圆形数字如下：

把一个十进制数转换为一个无符号二进制数，若该二进制数中 *0* 的个数大于或等于 *1* 的个数，则它就是一个圆形数字。

现在给定两个正整数 *a* 和 *b*，请问在区间 *[a,b]* 内有多少个圆形数字。

### 输入格式

输入占一行，包含两个整数 *a* 和 *b*。

### 输出格式

输出一个整数，表示圆形数字的个数。

### 数据范围

*1 ≤ a < b ≤ 2 × 10^9*

### 输入样例：

```
2 12
```

### 输出样例：

```
6
```
