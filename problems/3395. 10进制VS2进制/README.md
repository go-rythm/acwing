## [3395. 10进制VS2进制](https://www.acwing.com/problem/content/3398/)

### 题目

对于一个十进制数 *A*，将 *A* 转换为二进制数，然后按位逆序排列，再转换为十进制数 *B*，我们称 *B* 为 *A* 的二进制逆序数。

例如对于十进制数 *173*，它的二进制形式为 *10101101*，逆序排列得到 *10110101*，其十进制数为 *181*，*181* 即为 *173* 的二进制逆序数。

### 输入格式

一个十进制数 *A*。

### 输出格式

输出 *A* 的二进制逆序数 *B*。

### 数据范围

*A* 不超过 *1000* 位。

### 输入样例：

```
173
```

### 输出样例：

```
181
```
