## [3156. 卡片换位](https://www.acwing.com/problem/content/3159/)

### 题目

你玩过华容道的游戏吗？

这是个类似的，但更简单的游戏。

看下面 *3 x 2* 的格子

```
+---+---+---+
| A | * | * |
+---+---+---+
| B |   | * |
+---+---+---+
```

在其中放 *5* 张牌，其中 *A* 代表关羽，*B* 代表张飞， `*` 代表士兵。

还有一个格子是空着的。

你可以把一张牌移动到相邻的空格中去(对角不算相邻)。

游戏的目标是：关羽和张飞交换位置，其它的牌随便在哪里都可以。

### 输入格式

输入两行 *6* 个字符表示当前的局面。

### 输出格式

一个整数，表示最少多少步，才能把 *AB* 换位（其它牌位置随意）。

### 输入样例1：

```
* A
**B
```

### 输出样例1：

```
17
```

### 输入样例2：

```
A B
***
```

### 输出样例2：

```
12
```
