## [3656. 数字](https://www.acwing.com/problem/content/3659/)

### 题目

给定一个由 *0 ~ 9* 构成的长度为 *N* 的字符串，请你统计出现次数最多的数字。

### 输入格式

输入包含多组测试数据。

每组数据占一行，包含一个字符串。

### 输出格式

每组数据输出一行结果，表示出现最多的数字。

如果多个数字出现次数相同，则优先输出较小的数字。

### 数据范围

*1 ≤ N ≤ 1000*

### 输入样例：

```
1234567891
11122333
1235564
```

### 输出样例：

```
1
1
5
```
