## [3028. 最小圆覆盖](https://www.acwing.com/problem/content/3031/)

### 题目

在一个二维平面上给定 *N* 个点，请你画出一个最小的能够包含所有点的圆。

圆的边上的点视作在圆的内部。

### 输入格式

第一行包含一个整数 *N*。

接下来 *N* 行，每行包含两个实数，表示一个点的坐标 *(X_i,Y_i)*。

### 输出格式

共一行，输出圆的半径和圆心的坐标。

结果保留 *10* 位小数。

### 数据范围

*2 ≤ N ≤ 10^5*,

*-10000.0 ≤ X_i,Y_i ≤ 10000.0*

### 输入样例：

```
6
8.0 9.0
4.0 7.5
1.0 2.0
5.1 8.7
9.0 2.0
4.5 1.0
```

### 输出样例：

```
5.0000000000
5.0000000000 5.0000000000
```
