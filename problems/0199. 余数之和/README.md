## [199. 余数之和](https://www.acwing.com/problem/content/201/)

### 题目

给出正整数 *n* 和 *k*，计算 *j(n, k)=k bmod 1 + k bmod 2 + k bmod 3 + … + k bmod n* 的值。

例如 *j(5, 3)=3 bmod 1 + 3 bmod 2 + 3 bmod 3 + 3 bmod 4 + 3 bmod 5=0+1+0+3+3=7*。

### 输入格式

输入仅一行，包含两个整数 *n, k*。

### 输出格式

输出仅一行，即 *j(n, k)*。

### 数据范围

*1 ≤ n,k ≤ 10^9*

### 输入样例：

```
5 3
```

### 输出样例：

```
7
```
