## [1064\. 小国王](https://www.acwing.com/problem/content/1066/)

### 题目

在 $n \\times n$ 的棋盘上放 $k$ 个国王，国王可攻击相邻的 $8$ 个格子，求使它们无法互相攻击的方案总数。

### 输入格式

共一行，包含两个整数 $n$ 和 $k$。

### 输出格式

共一行，表示方案总数，若不能够放置则输出$0$。

### 数据范围

$1 \\le n \\le 10$,

$0 \\le k \\le n^2$

### 输入样例：

```
3 2
```

### 输出样例：

```
16
```
