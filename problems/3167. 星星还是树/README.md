## [3167. 星星还是树](https://www.acwing.com/problem/content/3170/)

### 题目

在二维平面上有 *n* 个点，第 *i* 个点的坐标为 *(x_i,y_i)*。

请你找出一个点，使得该点到这 *n* 个点的距离之和最小。

该点可以选择在平面中的任意位置，甚至与这 *n* 个点的位置重合。

### 输入格式

第一行包含一个整数 *n*。

接下来 *n* 行，每行包含两个整数 *x_i,y_i*，表示其中一个点的位置坐标。

### 输出格式

输出最小距离和，答案四舍五入取整。

### 数据范围

*1 ≤ n ≤ 100*,

*0 ≤ x_i,y_i ≤ 10000*

### 输入样例：

```
4
0 0
0 10000
10000 10000
10000 0
```

### 输出样例：

```
28284
```
