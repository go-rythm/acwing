## [3608. 大整数排序](https://www.acwing.com/problem/content/3611/)

### 题目

给定 *N* 个整数，将它们从小到大排序后输出。

### 输入格式

第一行包含一个整数 *N*。

接下来 *N* 行，每行包含一个整数。

### 输出格式

将所给整数从小达大排序后输出，每行输出一个整数。

### 数据范围

*1 ≤ N ≤ 100*,

输入整数均为正数，且长度不超过 *1000*。

### 输入样例：

```
3
11111111111111111111111111111
2222222222222222222222222222222222
33333333
```

### 输出样例：

```
33333333
11111111111111111111111111111
2222222222222222222222222222222222
```
