## [3634. IP地址](https://www.acwing.com/problem/content/3637/)

### 题目

输入一串字符，判断该字符串是否为点分十进制的 IP 地址，若是则转换为 *16* 进制输出，否则输出 `Error`。

### 输入格式

一行字符串。

注意，输入可能是任意的一个字符串，比如 `abc.bas.fefe.4r4` 或者 `23.23.11.23.123` 这都是不合法的 IP 地址。

### 输出格式

按题目要求，输出一行结果。

### 数据范围

输入字符串长度不超过 *100*。

### 输入样例1：

```
192.41.6.20
```

### 输出样例1：

```
0xC0290614
```

### 输入样例2：

```
257.32.23.1
```

### 输出样例2：

```
Error
```
