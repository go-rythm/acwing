## [866\. 试除法判定质数](https://www.acwing.com/problem/content/868/)

### 题目

给定 $n$ 个正整数 $a\_i$，判定每个数是否是质数。

### 输入格式

第一行包含整数 $n$。

接下来 $n$ 行，每行包含一个正整数 $a\_i$。

### 输出格式

共 $n$ 行，其中第 $i$ 行输出第 $i$ 个正整数 $a\_i$ 是否为质数，是则输出 `Yes`，否则输出 `No`。

### 数据范围

$1 \\le n \\le 100$,

$1 \\le a\_i \\le 2^{31}-1$

### 输入样例：

```
2
2
6
```

### 输出样例：

```
Yes
No
```
