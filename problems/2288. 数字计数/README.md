## [2288. 数字计数](https://www.acwing.com/problem/content/2290/)

### 题目

给定两个正整数 *a* 和 *b*，求在 *[a,b]* 中的所有整数中，每个数码(digit)各出现了多少次。

### 输入格式

仅包含一行两个整数 *a、b*，含义如上所述。

### 输出格式

包含一行 *10* 个整数，分别表示 *0-9* 在 *[a,b]* 中出现了多少次。

### 数据范围

*1 ≤ a ≤ b ≤ 10^{12}*

### 输入样例：

```
1 99
```

### 输出样例：

```
9 20 20 20 20 20 20 20 20 20
```
