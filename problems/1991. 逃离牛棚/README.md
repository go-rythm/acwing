## [1991. 逃离牛棚](https://www.acwing.com/problem/content/1993/)

### 题目

农夫约翰农场的挤奶时间到了，但是奶牛全部逃走了！

约翰需要把它们都抓回来，这需要你的帮助。

约翰的农场包含 *N* 个牧场，编号 *1 ~ N*，这些牧场由 *N-1* 条双向道路连接。

牛棚位于 *1* 号牧场，从牛棚可以到达任何其他牧场。

这些奶牛早上还在各自的农场中，现在谁知道它们跑哪里去了。

约翰能够确定的是，它们会朝 **远离牛棚** 的方向逃跑，并且最多只会跑出 *L* 单位距离。

对于每个牧场，约翰想知道从该牧场逃跑的奶牛最终可能会逃到多少个不同的牧场。

注意，奶牛可能懒得一动不动，还呆在最初所在的牧场，这也算是一种情况，应当予以计数。

### 输入格式

第 *1* 行：两个整数 *N* 和 *L*。

第 *2..N* 行：第 *i* 行包含两个整数 *p_i* 和 *l_i*， *p_i* 是牧场 *i* 和牛棚之间最短路径上的第一个牧场，而 *l_i* 是 *i* 与 *p_i* 之间道路的长度。

### 输出格式

共 *N* 行，每行包含一个数字，第 *i* 行的数字表示从牧场 *i* 沿远离牛棚的方向可以到达的相距不超过 *L* 的牧场数量（牧场 *i* 本身也要算上）。

### 数据范围

*1 ≤ N ≤ 2 × 10^5*,

*1 ≤ L ≤ 10^{18}*,

*1 ≤ p_i < i*,

*1 ≤ l_i ≤ 10^{12}*

### 输入样例：

```
4 5
1 4
2 3
1 5
```

### 输出样例：

```
3
2
1
1
```

### 样例解释

位于 *1* 号牧场的奶牛可以藏在 *1,2,4* 号牧场。

位于 *2* 号牧场的奶牛可以藏在 *2,3* 号牧场。

位于 *3* 号和 *4* 号牧场的奶牛无处可去，只能呆在各自的牧场。
