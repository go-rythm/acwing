## [3558. 整数和](https://www.acwing.com/problem/content/3561/)

### 题目

编写程序，读入一个整数 *N*。

若 *N* 为非负数，则计算 *N* 到 *2N* 之间的整数和；若 *N* 为一个负数，则计算 *2N* 到 *N* 之间的整数和（包括两端整数）。

### 输入格式

第一行包含整数 *T*，表示共有 *T* 组测试数据。

每组数据一行，一个整数 *N*。

### 输出格式

每组数据一行，一个结果。

### 数据范围

*1 ≤ T ≤ 100*,

*-100 ≤ N ≤ 100*

### 输入样例：

```
2
2
-1
```

### 输出样例：

```
9
-3
```
