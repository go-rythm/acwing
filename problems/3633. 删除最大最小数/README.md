## [3633. 删除最大最小数](https://www.acwing.com/problem/content/3636/)

### 题目

给定 *N* 个互不相同的数，请你删除其中最大和最小的数之后按原顺序将剩余的数输出。

### 输入格式

第一行包含整数 *N*。

第二行包含 *N* 个互不相同的整数。

### 输出格式

共一行，按原顺序输出删除最大和最小的数之后剩余的数。

### 数据范围

*3 ≤ N ≤ 100*,

输入整数取值范围 *[1,1000]*。

### 输入样例：

```
5
5 2 4 1 3
```

### 输出样例：

```
2 4 3
```
