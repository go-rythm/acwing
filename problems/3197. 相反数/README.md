## [3197. 相反数](https://www.acwing.com/problem/content/3200/)

### 题目

有 *N* 个非零且各不相同的整数。

请你编一个程序求出它们中有多少对相反数(*a* 和 *-a* 为一对相反数)。

### 输入格式

第一行包含一个正整数 *N*。

第二行为 *N* 个用单个空格隔开的非零整数，每个数的绝对值不超过 *1000*，保证这些整数各不相同。

### 输出格式

只输出一个整数，即这 *N* 个数中包含多少对相反数。

### 数据范围

*1 ≤ N ≤ 500*

### 输入样例：

```
5
1 2 3 -1 -2
```

### 输出样例：

```
2
```
