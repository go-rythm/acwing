## [2807. 中位数](https://www.acwing.com/problem/content/2809/)

### 题目

给出 *1 ~ n* 的一个排列，统计该排列有多少个长度为奇数的连续子序列的中位数是 *b*。

中位数是指把所有元素从小到大排列后，位于中间的数。

### 输入格式

第一行为两个正整数 *n* 和 *b*。

第二行为 *1 ~ n* 的排列。

### 输出格式

输出一个整数，即中位数为 *b* 的连续子序列个数。

### 数据范围

*1 ≤ n ≤ 10^5*,

*1 ≤ b ≤ n*

### 输入样例：

```
7 4
5 7 2 4 3 1 6
```

### 输出样例：

```
4
```

### 样例解释

*4* 个满足条件的连续子序列为：*{4}, {7,2,4}, {5,7,2,4,3}* 和 *{5,7,2,4,3,1,6}*。
