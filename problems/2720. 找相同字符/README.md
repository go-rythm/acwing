## [2720. 找相同字符](https://www.acwing.com/problem/content/2722/)

### 题目

给定两个字符串，求出在两个字符串中各取出一个子串使得这两个子串相同的方案数。

两个方案不同当且仅当这两个子串中有一个位置不同。

### 输入格式

两行，两个字符串 *s_1,s_2*，长度分别为 *n_1,n_2*。

### 输出格式

输出一个整数表示答案。

### 数据范围

*1 ≤ n_1,n_2 ≤ 2 × 10^5*,

字符串中只有小写字母。

### 输入样例：

```
aabb
bbaa
```

### 输出样例：

```
10
```
