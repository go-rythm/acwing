## [2801\. 三角形面积并](https://www.acwing.com/problem/content/2803/)

### 题目

给出 $n$ 个三角形，求它们并的面积。

### 输入格式

第一行为 $n$，即三角形的个数。

以下 $n$ 行，每行 $6$ 个实数 $x\_1, y\_1, x\_2, y\_2, x\_3, y\_3$，代表三角形的顶点坐标。

坐标均为不超过 $10 ^ 6$ 的实数，输入数据保留 $1$ 位小数。

### 输出格式

输出并的面积 $u$，保留两位小数。

### 数据范围

$1 \\le n \\le 100$

### 输入样例：

```
2
0.0 0.0 2.0 0.0 1.0 1.0
1.0 0.0 3.0 0.0 2.0 1.0
```

### 输出样例：

```
1.75
```
