## [890\. 能被整除的数](https://www.acwing.com/problem/content/892/)

### 题目

给定一个整数 $n$ 和 $m$ 个不同的质数 $p\_1, p\_2, …, p\_m$。

请你求出 $1 \\sim n$ 中能被 $p\_1, p\_2, …, p\_m$ 中的至少一个数整除的整数有多少个。

### 输入格式

第一行包含整数 $n$ 和 $m$。

第二行包含 $m$ 个质数。

### 输出格式

输出一个整数，表示满足条件的整数的个数。

### 数据范围

$1 \\le m \\le 16$,

$1 \\le n,p\_i \\le 10^9$

### 输入样例：

```
10 2
2 3
```

### 输出样例：

```
7
```
