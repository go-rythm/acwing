## [896\. 最长上升子序列 II](https://www.acwing.com/problem/content/898/)

### 题目

给定一个长度为 $N$ 的数列，求数值严格单调递增的子序列的长度最长是多少。

### 输入格式

第一行包含整数 $N$。

第二行包含 $N$ 个整数，表示完整序列。

### 输出格式

输出一个整数，表示最大长度。

### 数据范围

$1 \\le N \\le 100000$，

$-10^9 \\le 数列中的数 \\le 10^9$

### 输入样例：

```
7
3 1 2 1 8 5 6
```

### 输出样例：

```
4
```
