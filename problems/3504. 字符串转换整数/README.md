## [3504. 字符串转换整数](https://www.acwing.com/problem/content/3507/)

### 题目

给定一个字符串，字符串由数字和大小写字母构成，请你找到并输出其中的 **第一个** 有效整数。

如果无法找到有效整数，或者找到的有效整数超过 int 范围，则输出 *-1*。

### 输入格式

一行一个字符串。

### 输出格式

一个整数表示结果。

### 数据范围

字符串长度不超过 *100*

### 输入样例1：

```
2016
```

### 输入样例1：

```
2016
```

### 输入样例2：

```
o627CSo1
```

### 输出样例2：

```
627
```

### 输入样例3：

```
123456789123abc
```

### 输出样例3：

```
-1
```

### 输入样例4：

```
abc
```

### 输出样例4：

```
-1
```
