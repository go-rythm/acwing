## [3501. 后缀子串排序](https://www.acwing.com/problem/content/3504/)

### 题目

对于一个字符串，将其后缀子串进行排序。

例如 `grain` 其子串有： `grain rain ain in n`。

然后对各子串按字典顺序排序，即： `ain grain in n rain`。

### 输入格式

一行，一个由小写字母构成的字符串。

### 输出格式

将排好序的子串依次输出，每行输出一个。

### 数据范围

输入字符串长度不超过 *1000*。

### 输入样例：

```
grain
```

### 输出样例：

```
ain
grain
in
n
rain
```
