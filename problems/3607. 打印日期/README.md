## [3607. 打印日期](https://www.acwing.com/problem/content/3610/)

### 题目

给出年分 *y* 和一年中的第 *d* 天，算出第 *d* 天是几月几号。

### 输入格式

输入包含多组测试数据。

每组数据占一行，包含两个整数 *y* 和 *d*。

### 输出格式

每组数据输出一行一个结果，格式为 `yyyy-mm-dd`。

### 数据范围

输入最多包含 *100* 组数据,

*1 ≤ y ≤ 3000*,

*1 ≤ d ≤ 366*,

数据保证合法。

### 输入样例：

```
2000 3
2000 31
2000 40
2000 60
2000 61
2001 60
```

### 输出样例：

```
2000-01-03
2000-01-31
2000-02-09
2000-02-29
2000-03-01
2001-03-01
```
