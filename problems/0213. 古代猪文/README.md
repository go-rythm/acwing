## [213. 古代猪文](https://www.acwing.com/problem/content/215/)

### 题目

给定整数 *n,q*，计算 *q^{sum_{d|n}C^d_n}bmod999911659*。

### 输入格式

输入包括一行，包含两个整数 *n，q*，用一个空格隔开。

### 输出格式

输出包括一行，包含一个整数表示最终结果。

### 数据范围

*1 ≤ n,q ≤ 10^9*

### 输入样例：

```
4 2
```

### 输出样例：

```
2048
```

**提示**:对于 *n* 的每一个正因数 *d*，都有一个 *C^d_n* 的值，将它们全部加起来得到的和就是 *sum_{d|n}C^d_n*。
