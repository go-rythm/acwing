## [2991. 弦论](https://www.acwing.com/problem/content/2994/)

### 题目

对于一个给定长度为 *N* 的字符串，求它的第 *K* 小子串是什么。

### 输入格式

第一行是一个仅由小写英文字母构成的字符串 *S*。

第二行为两个整数 *T* 和 *K*，*T = 0* 则表示不同位置的相同子串算作一个。*T=1* 则表示不同位置的相同子串算作多个。*K* 的意义如题所述。

### 输出格式

输出仅一行，为一个数字串，为第 *K* 小的子串。

如果子串数目不足 *K* 个，则输出 *-1*。

### 数据范围

*1 ≤ N ≤ 5 × 10^5*,

*1 ≤ K ≤ 10^9*

### 输入样例：

```
aabc
0 3
```

### 输出样例：

```
aab
```
