## [3709. 单链表节点交换](https://www.acwing.com/problem/content/3712/)

### 题目

输入一个单链表，将奇数位置上的节点与偶数位置上的节点进行交换，然后输出交换之后的链表。

### 输入格式

第一行包含整数 *n*，表示节点数量。

第二行包含 *n* 个整数，表示每个节点的值。

### 输出格式

共一行，输出交换后的链表。

### 数据范围

*1 ≤ n ≤ 100*,

节点取值范围 *[1,100]*。

### 输入样例：

```
5
1 2 3 4 5
```

### 输出样例：

```
2 1 4 3 5
```
