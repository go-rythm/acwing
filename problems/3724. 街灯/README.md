## [3724. 街灯](https://www.acwing.com/problem/content/3727/)

### 题目

现在是基督降临节，在长为 *N* 米的街道上有 *M* 个街灯。

每个灯照亮了左边 *K* 米，右边 *K* 米。

也就是说，在 *X* 米处的灯，能照亮从 *X-K* 到 *X+K*（含）。

当然，街道某处可能被多个灯照亮。

所有灯位于不同的位置。

问题在于有可能这些灯没法照亮整条街道。

你的任务是，确定最少还要加多少灯，使得整条街道都被照亮。

### 输入格式

输入包含多组测试数据。

每组数据第一行包含三个整数 *N,M,K*.

第二行包含 *M* 个升序的整数，表示每个灯的位置。

### 输出格式

每组数据输出一行，一个整数，表示答案。

### 数据范围

*1 ≤ N ≤ 1000*,

*1 ≤ M ≤ N*,

*0 ≤ K ≤ N*,

输入最多包含 *100* 组数据。

### 输入样例：

```
5 2 2
1 5
5 1 2
2
```

### 输出样例：

```
0
1
```
