## [3657. 防水堤坝](https://www.acwing.com/problem/content/3660/)

### 题目

在太平洋的一个小岛上，岛民想要建立一个环岛的堤坝，我们可以将小岛简化为一个二维平面，你需要使用 *K* 条边（这些边要么是水平或者垂直长度为 *1* 的边，要么是 *45* 度倾斜的长度为 *sqrt{2}* 的边）围成一个多边形，多边形的顶点必须位于整点，然后要让围成的多边形面积最大，你需要求出最大面积是多少。

### 输入格式

输入包含多组测试数据。

每组数据包含一个整数 *K*。

### 输出格式

每组数据输出一行答案，结果保留一位小数。

### 数据范围

*3 ≤ K ≤ 2 × 10^9*

### 输入样例：

```
3
4
5
6
```

### 输出样例：

```
0.5
2.0
2.5
4.0
```
