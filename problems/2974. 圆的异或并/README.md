## [2974. 圆的异或并](https://www.acwing.com/problem/content/2977/)

### 题目

在平面直角坐标系中给定 *N* 个圆。

已知这些圆两两没有交点，即两圆的关系只存在相离和包含。

求这些圆的异或面积并。

异或面积并为：当一片区域在奇数个圆内则计算其面积，当一片区域在偶数个圆内则不考虑。

### 输入格式

第一行包含一个正整数 *N*，代表圆的个数。

接下来 *N* 行，每行 *3* 个非负整数 *x,y,r*，表示一个圆心在 *(x,y)*，半径为 *r* 的圆。

### 输出格式

仅一行一个整数，表示所有圆的异或面积并除以圆周率 *pi* 的结果。

### 数据范围

*1 ≤ N ≤ 2 × 10^5*,

*|x|,|y| ≤ 10^8*，

*0 < r < 10^8*

### 输入样例：

```
2
0 0 1
0 0 2
```

### 输出样例：

```
3
```
