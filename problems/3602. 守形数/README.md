## [3602. 守形数](https://www.acwing.com/problem/content/3605/)

### 题目

守形数是这样一种整数，它的平方的低位部分等于它本身。

比如 *25* 的平方是 *625*，低位部分是 *25*，因此 *25* 是一个守形数。

编一个程序，判断 *N* 是否为守形数。

### 输入格式

题目包含若干组数据。

每组数据占一行，包含一个 *N*。

### 输出格式

每组数据输出一行，一个结果，如果 *N* 是守形数，则输出 `Yes!`，否则输出 `No!`。

### 数据范围

*2 ≤ N < 100*

### 输入样例：

```
25
4
```

### 输出样例：

```
Yes!
No!
```
