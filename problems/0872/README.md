## [872\. 最大公约数](https://www.acwing.com/problem/content/874/)

### 题目

给定 $n$ 对正整数 $a\_i, b\_i$，请你求出每对数的最大公约数。

### 输入格式

第一行包含整数 $n$。

接下来 $n$ 行，每行包含一个整数对 $a\_i,b\_i$。

### 输出格式

输出共 $n$ 行，每行输出一个整数对的最大公约数。

### 数据范围

$1 \\le n \\le 10^5$,

$1 \\le a\_i, b\_i \\le 2 \\times 10^9$

### 输入样例：

```
2
3 6
4 6
```

### 输出样例：

```
3
2
```
