## [3169. 等腰三角形](https://www.acwing.com/problem/content/3172/)

### 题目

本题目要求你输出一个由数字组成的等腰三角形。

具体的步骤是：

1. 先用 *1,2,3,…* 的自然数拼一个足够长的串。
2. 用这个串填充三角形的三条边。从上方顶点开始，逆时针填充。

比如，当三角形高度是 *8* 时：

```
       1
      2 1
     3   8
    4     1
5       7
6         1
7           6
891011121314151
```

### 输入格式

一个正整数 *n*，表示三角形的高度。

### 输出格式

输出，用数字填充的等腰三角形。

为了便于测评，我们要求空格一律用 `.` 代替。

具体，可参照样例。

### 数据范围

*3 < n < 300*

### 输入样例1：

```
5
```

### 输出样例1：

```
....1
...2.1
..3...2
.4.....1
567891011
```

### 输入样例2：

```
10
```

### 输出样例2：

```
.........1
........2.2
.......3...2
......4.....2
.....5.......1
....6.........2
...7...........0
..8.............2
.9...............9
1011121314151617181
```

### 输入样例3：

```
15
```

### 输出样例3：

```
..............1
.............2.3
............3...2
...........4.....3
..........5.......1
.........6.........3
........7...........0
.......8.............3
......9...............9
.....1.................2
....0...................8
...1.....................2
..1.......................7
.1.........................2
21314151617181920212223242526
```
