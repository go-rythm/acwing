## [3638. 排序](https://www.acwing.com/problem/content/3641/)

### 题目

输入 *n* 个数进行排序，要求先按奇偶后按从小到大的顺序排序。

### 输入格式

第一行包含一个整数 *n*。

第二行包含 *n* 个整数。

### 输出格式

输出排序之后的结果。

整数之间空格隔开。

### 数据范围

*1 ≤ n ≤ 1000*,

输入整数取值范围 *[1,10000]*。

### 输入样例：

```
8
1 2 3 4 5 6 7 8
```

### 输出样例：

```
1 3 5 7 2 4 6 8
```
