## [3545. 寻找奇特的数](https://www.acwing.com/problem/content/3548/)

### 题目

给定一个 *n × n* 的矩阵 *a*，其中第 *i* 行第 *j* 列的元素 *a_{ij}* 等于 *i × j*（*1 ≤ i,j ≤ n*）。

现在，给定一个整数 *m*，请你计算 *m* 在矩阵中出现的次数。

### 输入格式

第一行一个整数 *T*，表示共有 *T* 组测试数据。

每组数据包含两个整数 *n,m*。

### 输出格式

每组数据输出一行一个结果，表示 *m* 在矩阵中出现的次数。

### 数据范围

*1 ≤ T ≤ 20*,

*1 ≤ n ≤ 10^5*,

*1 ≤ m ≤ 10^9*

### 输入样例：

```
2
3 6
3 3
```

### 输出样例：

```
2
2
```
