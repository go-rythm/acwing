## [3606. A+B](https://www.acwing.com/problem/content/3609/)

### 题目

给定两个整数 *A* 和 *B*，其表示形式是：从个位开始，每三位数用逗号 `,` 隔开。 现在请计算 *A+B* 的结果，并以正常形式输出。

### 输入格式

输入包含多组数据数据。

每组数据占一行，由两个整数 *A* 和 *B* 组成。

### 输出格式

请计算 *A+B* 的结果，并以正常形式输出，每组数据占一行。

### 数据范围

每个输入最多包含 *100* 组数据。

*-10^9 < A,B < 10^9*。

### 输入样例：

```
-234,567,890 123,456,789
1,234 2,345,678
```

### 输出样例：

```
-111111101
2346912
```
