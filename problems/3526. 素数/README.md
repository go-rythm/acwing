## [3526. 素数](https://www.acwing.com/problem/content/3529/)

### 题目

输入一个整数 *n*，要求输出所有从 *1* 到这个整数之间(不包括 *1* 和这个整数)个位为 *1* 的素数，如果没有则输出 *-1*。

### 输入格式

输入包含多组测试数据。

每组数据占一行，包含一个整数 *n*。

### 输出格式

每组数据输出占一行，由小到大输出所有满足条件的素数，素数之间单个空格隔开。如果没有则输出 *-1*。

### 数据范围

*2 ≤ n ≤ 10000*,

输入最多包含 *100* 组数据。

### 输入样例：

```
100
```

### 输出样例：

```
11 31 41 61 71
```
