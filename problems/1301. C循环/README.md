## [1301. C循环](https://www.acwing.com/problem/content/1303/)

### 题目

对于 C 语言的循环语句，形如：

```
for (variable = A; variable != B; variable += C)
statement;
```

请问在 *k* 位存储系统中循环几次才会结束。

若在有限次内结束，则输出循环次数。否则输出死循环。

### 输入格式

多组数据，每组数据一行四个整数 *A,B,C,k*。

读入以 0 0 0 0 结束。

### 输出格式

若在有限次内结束，则输出循环次数。

否则输出 FOREVER。

### 数据范围

*1 ≤ k ≤ 32*,

*0 ≤ A,B,C < 2^k*

### 输入样例：

```
3 3 2 16
3 7 2 16
7 3 2 16
3 4 2 16
0 0 0 0
```

### 输出样例：

```
0
2
32766
FOREVER
```
