## [3702. 十六进制不进位加法](https://www.acwing.com/problem/content/3705/)

### 题目

*16* 进制不进位的加法，即和正常加法类似，只是不用去计算进位的数，比正常的加法更简单。

如 *A + 6 = 0*（正常加法是 *10*，但是由于不进位所以只有 *0*）。

### 输入格式

输入包含多组测试数据。

每组数据占一行，包含两个十六进制数，字母统一大写。

### 输出格式

每组数据输出一行不进位加法的结果。

### 数据范围

输入数字长度不超过 *100*。

### 输入样例：

```
123 456
6 A
```

### 输出样例：

```
579
0
```
