## [1498. 最深的根](https://www.acwing.com/problem/content/1500/)

### 题目

一个无环连通图可以被视作一个树。

树的高度取决于所选取的根节点。

现在，你要找到可以使得树的高度最大的根节点。

它被称为最深的根。

### 输入格式

第一行包含整数 *N*，表示节点数量。

节点编号为 *1 ~ N*。

接下来 *N-1* 行，每行包含两个整数，表示两个节点之间存在一条边。

### 输出格式

输出最深的根的节点编号。

如果最深的根不唯一，则按照从小到大的顺序，将它们依次输出，每个占一行。

如果给定的图不是树，输出 `Error: K components`，其中 *K* 是图中连通分量的数量。

### 数据范围

*1 ≤ N ≤ 10^4*

### 输入样例1：

```
5
1 2
1 3
1 4
2 5
```

### 输出样例1：

```
3
4
5
```

### 输入样例2：

```
5
1 3
1 4
2 5
3 4
```

### 输出样例2：

```
Error: 2 components
```
