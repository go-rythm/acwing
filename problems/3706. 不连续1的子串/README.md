## [3706. 不连续1的子串](https://www.acwing.com/problem/content/3709/)

### 题目

请计算长度为 *N* 且不含连续 *1* 的 *01* 串的个数。

例如，当 *N = 3* 时，答案为 *5*，因为长度为 *3* 且不含连续 *1* 的 *01* 串一共 *5* 个：*000, 001, 010, 100, 101*。

### 输入格式

一个整数 *N*。

### 输出格式

一个整数，表示结果。

### 数据范围

*1 ≤ N ≤ 20*

### 输入样例：

```
3
```

### 输出样例：

```
5
```
