## [445. 数字反转](https://www.acwing.com/problem/content/447/)

### 题目

给定一个整数，请将该数各个位上数字反转得到一个新数。

新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零。

### 输入格式

输入共 *1* 行，*1* 个整数 *N*。

### 输出格式

输出共 *1* 行，*1* 个整数表示反转后的新数。

### 数据范围

*|N| ≤ 10^9*

### 输入样例：

```
123
```

### 输出样例：

```
321
```

### 输入样例：

```
-380
```

### 输出样例：

```
-83
```
