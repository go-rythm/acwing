## [97. 约数之和](https://www.acwing.com/problem/content/99/)

### 题目

假设现在有两个自然数 *A* 和 *B*，*S* 是 *A^B* 的所有约数之和。

请你求出 *S bmod 9901* 的值是多少。

### 输入格式

在一行中输入用空格隔开的两个整数 *A* 和 *B*。

### 输出格式

输出一个整数，代表 *S bmod 9901* 的值。

### 数据范围

*0 ≤ A,B ≤ 5 × 10^7*

### 输入样例：

```
2 3
```

### 输出样例：

```
15
```

**注意**: *A* 和 *B* 不会同时为 *0*。
