## [1571. 完美序列](https://www.acwing.com/problem/content/1573/)

### 题目

给定一个正整数序列和一个正整数 *p*。

如果 *M ≤ m × p* 成立，则该序列被称为完美序列，其中 *M* 和 *m* 分别是序列中的最大和最小数。

现在给定一个序列和一个参数 *p*，你应该从序列中找到尽可能多的数字以构成一个完美的子序列。

### 输入格式

第一行包含两个整数 *N* 和 *p*。

第二行包含 *N* 个正整数，表示给定序列。

### 输出格式

输出最多可以选择多少个数，构成一个完美子序列。

### 数据范围

*1 ≤ N ≤ 10^5*,

*1 ≤ p ≤ 10^9*,

序列中的元素均不超过 *10^9*。

### 输入样例：

```
10 8
2 3 20 4 5 1 6 7 8 9
```

### 输出样例：

```
8
```
