## [1100\. 抓住那头牛](https://www.acwing.com/problem/content/1102/)

### 题目

农夫知道一头牛的位置，想要抓住它。

农夫和牛都位于数轴上，农夫起始位于点 $N$，牛位于点 $K$。

农夫有两种移动方式：

1. 从 $X$ 移动到 $X-1$ 或 $X+1$，每次移动花费一分钟
2. 从 $X$ 移动到 $2\*X$，每次移动花费一分钟

假设牛没有意识到农夫的行动，站在原地不动。

农夫最少要花多少时间才能抓住牛？

### 输入格式

共一行，包含两个整数N和K。

### 输出格式

输出一个整数，表示抓到牛所花费的最少时间。

### 数据范围

$0 \\le N,K \\le 10^5$

### 输入样例：

```
5 17
```

### 输出样例：

```
4
```
