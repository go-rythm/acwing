## [3393. 最大序列和](https://www.acwing.com/problem/content/3396/)

### 题目

给出一个整数序列 *S*，其中有 *N* 个数，定义其中一个非空连续子序列 *T* 中所有数的和为 *T* 的“序列和”。

对于 *S* 的所有非空连续子序列 *T*，求最大的序列和。

### 输入格式

第一行包含一个整数 *N*。

第二行包含 *N* 个整数，表示序列中的元素。

### 输出格式

输出一个数，表示最大序列和。

### 数据范围

*1 ≤ N ≤ 10^6*,

序列中的元素的取值范围 *[-10^9,10^9]*。

### 输入样例1：

```
5
1 5 -3 2 4
```

### 输出样例1：

```
9
```

### 输入样例2：

```
6
1 -2 3 4 -10 6
```

### 输出样例2：

```
7
```

### 输入样例3：

```
4
-3 -1 -2 -5
```

### 输出样例3：

```
-1
```
