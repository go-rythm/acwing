## [2873. 平面切分](https://www.acwing.com/problem/content/2876/)

### 题目

平面上有 *N* 条直线，其中第 *i* 条直线是 *y = A_i · x + B_i*。

请计算这些直线将平面分成了几个部分。

### 输入格式

第一行包含一个整数 *N*。

以下 *N* 行，每行包含两个整数 *A_i, B_i*。

### 输出格式

一个整数代表答案。

### 数据范围

对于 *50%* 的评测用例，*1 ≤ N ≤ 4, −10 ≤ A_i, B_i ≤ 10*。

对于所有评测用例，*1 ≤ N ≤ 1000, −100000 ≤ A_i, B_i ≤ 100000*。

### 输入样例：

```
3
1 1
2 2
3 3
```

### 输出样例：

```
6
```
