## [3074\. 自适应辛普森积分](https://www.acwing.com/problem/content/3077/)

### 题目

给定两个整数 $a,b$，请计算如下积分：

$$\\int\_a^b \\frac{Sin(x)}{x}dx$$

### 输入格式

共一行，包含两个实数 $a,b$。

### 输出格式

输出一个实数，表示结果。

结果保留 $6$ 位小数。

### 数据范围

$1 \\le a < b \\le 100$

### 输入样例：

```
1.0 2.0
```

### 输出样例：

```
0.659330
```
